# ARCH Technologies Cybersecurity Tasks

## Task 1: Basic Network Sniffer
A Python-based network sniffer that captures and analyzes network traffic packets.

Features:
- Captures IP packets
- Extracts source and destination IP
- Detects protocol type (TCP, UDP, ICMP)
- Basic suspicious behavior detection
- Simple port scan detection logic

## Task 2: Keylogging Simulation (Controlled Environment Only)
A basic keylogger simulation script developed strictly for educational and defensive cybersecurity research purposes.

Features:
- Captures local keystrokes
- Logs to a local file
- Demonstrates how keystroke logging attacks work
- Includes risk analysis documentation

⚠️ This project was developed strictly for cybersecurity education and awareness purposes.
